1. Project Overview
1.1 Introduction
AI-Powered Stock Screener is an end-to-end intelligent financial query platform that enables users to
interact with stock market and company financial data using natural language.


Instead of manually writing SQL queries or using rigid dashboard filters, users can simply type queries
such as:


     • "Technology companies with PE ratio less than 20"
     • "Companies with revenue growth in Q2 2024"
     • "Companies with high revenue growth and low PE ratio"

The system processes the query through multiple backend layers:


    1. Natural Language Understanding Layer (LLM)
    2. DSL Translation Layer
    3. DSL Validation Layer
    4. SQL Compilation Layer
    5. Query Execution Layer
    6. Result Formatting Layer
    7. Frontend Presentation Layer

The system was designed as a modular architecture to ensure:


     • Scalability
     • Security
     • Query Safety
     • Extensibility
     • Maintainability
     • AI integration capability

5.5 DSL Layer
Purpose
DSL (Domain Specific Language) acts as an intermediate representation between natural language and
SQL.




Why DSL?
Direct NL-to-SQL is risky because:


     • SQL injection risk
     • Uncontrolled query generation
     • Invalid queries
     • Schema hallucinations
     • Poor maintainability

DSL solves these by introducing a controlled abstraction layer.




                                                    9
DSL Structure

  {
      "filters": [
        {
          "field": "pe_ratio",
            "operator": "<",
            "value": 20
        }
      ],
      "logic": "AND"
  }




Extended DSL for Time Filtering

  {
      "filters": [
         {
           "field": "revenue",
           "operator": ">",
           "value": 100000
         }
      ],
      "logic": "AND",
      "time_filter": {
         "type": "quarter",
         "value": "2024-Q2"
      }
  }
